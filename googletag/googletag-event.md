# 事件

## googletag.events.SlotRenderEndedEvent

在廣告顯示後觸發的事件

```javascript
googletag.pubads().addEventListener('slotRenderEnded', function(event) {
  console.log('Slot has been rendered:');
  console.log(event);
});
```
